Laptop Prices

This dataset is a postprocessed version of the famous Laptop Price dataset, providing a comprehensive collection of information on various laptops, enabling detailed analysis of their specifications and pricing.

Source: Laptop Prices Dataset on Kaggle

Dataset Columns

Column Name Data Type Description
Company String Laptop Manufacturer
Product String Brand and Model
TypeName String Type (e.g., Notebook, Ultrabook, Gaming)
Inches Float Screen Size in Inches
ScreenResolution String Screen Resolution
Cpu String CPU Model
Ram String RAM Size (e.g., 8GB)
Memory String Storage Capacity and Type
Gpu String GPU Model
OpSys String Operating System
Weight String Weight of the Laptop
Price_euros Float Price in Euros
