Laptop Prices

This dataset is a refined version of the well-known Laptop Price dataset, designed to offer a comprehensive collection of information on various laptops, facilitating detailed analysis of their specifications and pricing. It consists of 23 columns containing both numerical and categorical values. The original dataset was highly compact, with extensive details embedded within each column. While this format was human-readable and concise, it was not optimal for Machine Learning algorithms, which perform better when distinct attributes are stored in separate columns. To enhance efficiency, the dataset was restructured by extracting individual details into separate fields. As a result, 28 duplicate rows were identified and removed, leading to the finalized version presented here. The original dataset has 1270 rows but what we have here is 500 random samples from it. Moreover, from the 23 columns, the categorical columns with only Yes or No values were dropped. The dataset is available in CSV format and can be downloaded from the link below.

Source: Laptop Prices Dataset on Kaggle

In here we have histogram for numerical variables, bar chart for categorical variables, and scatter plot for numerical variables. You can choose the X and Y variables from the dropdowns below. The plots can be toggled to be sideway and reset to its original position. For categories more than 10 items, we only show the first 10 items. For scatter plot, we are using jitter when having a categorical variable. But for the numerical values we are using the actual values.
